Project Overview

Have you ever had to fill out the same thing again and again on a website like Salesforce, only to have to change one field each time? Where Chrome Auto fill just isn't enough?

Then Chrome-Automation is a Chrome Extension tool for you! You set record on a website to begin to "learn" what things you do/don't do, then over a short period an automated script will be generated by the chrome extension. Think of them as your own personalised Chrome Extentions. You will be able to turn them on/off as desired and share the scripts with colleagues!

Motivation

My motivation to create Chrome-Automation is simple; I hate laborious repetitive tasks. I currently use a tool called Tampermonkey to allow me to create simple, to very complicated JavaScript snippets to automate part of my work. Creating these scripts does take some time, and the Chrome-Automation would be focused on machine learning to automate this part of the task, which can involve a lot of effort.

I often want to share these scripts with others, and want others to have the benefit of these simple types of micro-automation without the need be proficient in CSS/JS/HTML.

I believe that an array of small micro-automation tools will have an overall and holistic positive impact on a user's efficiency, accuracy and sanity.

Project Description

Tool Overview

The tool would feature as a Chrome Extension. Users install the chrome extension then "Record" what they would like to automate. Using Machine Learning, over several attempts at a user competing an action Chrome-Automation would generate a script to automate that task.

Features

Learning
Machine learning is a method of data analysis that automates analytical model building. It is a branch of artificial intelligence based on the idea that systems can learn from data, identify patterns and make decisions with minimal human intervention.
Using this technology, the tool will create a set of scripts to automate basic daily tasks within your websites or web-apps.

Multi-Website Functionality
As the extension is localised to a single machine then we can bypass some of the Internet browser rules around cross page scripting to allow a plethora of new possibilities.

  • Storing information to be used later, or on another website
  • Pull information from one website to another
  • Share information from your current website/web-apps and use APIs or Web-forms to trigger events without switching to the chrome tab, or even opening the website!

Re-arrange a website/web-app in Editor mode
If you like a button, link, or other object in a different place, or it impacts the work flow, then you will be able to move the object to a new desired location using “Drag-and-Drop"

Enterprise services

  • Auto-deploy scripts on employees managed computers
  • Use the bulk “learning” of all people in the organisation to generate the automated scripts faster
  • Ability to create an Enterprise Library so that only those within the organisation can access the scripts

Library of existing scripts
A Cloud-based library of existing scripts created by others of the community that allow users import, and share resources online. This will also autorise cloud storage and auto-updates of subscribed scripts.

Community space
A space where Community members can contribute, comment, start discussions and ask questions and, create new ideas.
Ideas - A space where all clients can suggest changes to the service which can be voted upon by the community to dedicate development resources.

Service

As a means to generate income from this service I would suggest the Freemium SaaS model. There would be several tiers of this service from Enterprise to induvial use. An example of the types of Tiered service would be;

  • Tier 1 - Enterprise
    • Unlimited script storage
    • Faster processing on the conversion of scripts, means less repeated actions from the user before an automated script was generated
    • A plain-text access to scripts for minor editing and saving
    • Ability to share and enable scripts across the organisation
    • Unlimited Library Access
    • Ability to vote on community ideas or submit new ones, as an organisation or individually
  • Tier 2 - Premium
    • Unlimited script storage
    • Faster script learning processing time
    • Unlimited Library access
    • Ability to vote on community ideas and submit new ones
  • Tier 3 - Free
    • A limited number of stored scripts
    • A limited number of synced library items
    • Can comment on existing ideas, limited votes, but cannot create new ideas
